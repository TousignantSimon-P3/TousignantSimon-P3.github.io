    Exercices de la Semaine 1 : Les Bases

    Ces exercices sont conçus pour pratiquer les concepts de base vus dans les notes de la semaine 1.
    Concentrez-vous sur la maîtrise de la compilation, de l'affichage, de la saisie et des fonctions simples.

    Bloc 1 : Affichage et Saisie Simple

    Exercice 1 : Bonjour le Monde - Objectif : Créer un programme C++ de base.
    - Tâche : Écrivez un programme qui affiche "Bonjour le monde !" dans la console et qui se termine en retournant 0.

    Exercice 2 : Fiche de Présentation - Objectif : Manipuler std::cout avec plusieurs lignes.
    - Tâche : Écrivez un programme qui affiche votre nom, votre ville et votre âge sur des lignes séparées.

    Exercice 3 : Déclaration de Variables - Objectif : Se familiariser avec l'initialisation uniforme.
    - Tâche : Déclarez quatre variables en utilisant la syntaxe d'accolades {} : un int, un double, un char, et un bool.
    Assignez-leur des valeurs de votre choix et affichez chaque valeur avec une description.

    Exercice 4 : Interaction Simple - Objectif : Utiliser std::cin >> avec std::string et int.
    - Tâche : Demandez à l'utilisateur son prénom et son année de naissance.
    Calculez (approximativement) son âge et affichez un message comme "Bonjour [Prénom], vous avez environ [Age] ans.".

    Exercice 5 : Calcul d'Aire - Objectif : Utiliser des types double avec std::cin.
    - Tâche : Demandez à l'utilisateur la largeur et la hauteur d'un rectangle.
    Calculez et affichez son aire.

    Bloc 2 : Gestion du Buffer et Erreurs de Saisie

    Exercice 6 : Le Piège de getline - Objectif : Résoudre le conflit entre std::cin >> et std::getline.
    - Tâche : 1. Demandez à l'utilisateur son numéro de porte (un int).
    2. Demandez-lui ensuite son nom de rue complet (un std::string avec getline).
    3. Observez que le programme ignore la saisie de la rue.
    4. Corrigez le problème en nettoyant le buffer après la lecture du nombre.

    Exercice 7 : Gestion d'Erreur Simple - Objectif : Utiliser std::cin.fail() pour détecter une erreur.
    - Tâche : Demandez à l'utilisateur d'entrer un nombre. Si la saisie échoue (si l'utilisateur tape du texte), affichez "Erreur : saisie invalide.". Sinon, affichez "Merci !".

    Exercice 8 : Boucle de Saisie Robuste - Objectif : Créer une boucle qui force une saisie numérique valide.
    - Tâche : Créez une boucle while qui continue de demander "Entrez votre age :" jusqu'à ce que l'utilisateur entre un nombre entier valide. Si l'utilisateur se trompe, le programme doit afficher une erreur et lui redemander.

    Exercice 9 : Analyse d'Entrée Mixte - Objectif : Comprendre comment cin gère les lectures partielles.
    - Tâche : 1. Demandez à l'utilisateur un nombre. L'utilisateur doit taper 45abc.
    2. Lisez la saisie dans une variable de type int.
    3. Affichez l'entier que vous avez réussi à lire.
    4. Ensuite, lisez le reste du buffer dans une variable std::string.
    5. Affichez la chaîne de caractères. Le résultat devrait vous montrer ce qui restait dans le buffer.

    Bloc 3 : Fonctions et Tableaux (Classique)

    Exercice 10 : Calculatrice Simple - Objectif : Écrire et appeler des fonctions simples.
    - Tâche : Écrivez quatre fonctions : addition(int a, int b), soustraction(int a, int b), multiplication(int a, int b), et division(int a, int b).
    Chaque fonction doit retourner le résultat du calcul. Dans main, testez chaque fonction avec des valeurs de votre choix et affichez les résultats.

    Exercice 11 : Passage par Copie - Objectif : Constater l'effet d'un passage par copie.
    - Tâche : 1. Écrivez une fonction ajouter_vingt(int nombre) qui reçoit un entier, lui ajoute 20, et n'a pas de return.
    2. Dans main, initialisez une variable ma_valeur {10}.
    3. Appelez ajouter_vingt(ma_valeur).
    4. Affichez ma_valeur après l'appel. A-t-elle changé ? Pourquoi ?

    Exercice 12 : Passage par Référence - Objectif : Utiliser une référence pour modifier une variable externe.
    - Tâche : 1. Écrivez une fonction reinitialiser(int& nombre) qui change la valeur de l'entier reçu à 0.
    2. Dans main, initialisez une variable ma_valeur {123}.
    3. Appelez reinitialiser(ma_valeur).
    4. Affichez ma_valeur après l'appel. A-t-elle changé ? Pourquoi ?

    Exercice 13 : Somme d'un Tableau (Style C) - Objectif : Passer un tableau à une fonction (méthode classique).
    - Tâche : Créez une fonction calculer_somme(const int tableau[], int taille) qui retourne la somme de tous les éléments d'un tableau.
    Dans main, initialisez un tableau (ex: int tableau[] {10, 20, 30};), trouvez sa taille avec std::ssize, et affichez la somme retournée par votre fonction.

    Exercice 14 : Trouver le Maximum - Objectif : Parcourir un tableau dans une fonction.
    - Tâche : Créez une fonction trouver_max(const int tableau[], int taille) qui retourne la plus grande valeur d'un tableau. Testez-la dans main.

    Exercice 15 : Inversion de Tableau - Objectif : Modifier un tableau via une fonction.
    - Tâche : Créez une fonction inverser_tableau(int tableau[], int taille) qui inverse les éléments du tableau sur place (le premier devient le dernier, etc.).
    Dans main, déclarez un tableau comme int tableau[] {1, 2, 3, 4, 5};, affichez-le, appelez la fonction, puis affichez-le à nouveau pour voir le changement.

    Bloc 4 : C++ Moderne (std::span & Vector)

    Exercice 16 : La Règle d'Or (std::span) - Objectif : Remplacer le couple (tableau, taille) par std::span.
    - Tâche : 1. Reprenez votre fonction calculer_somme de l'exercice 13. 2. Changez sa signature pour qu'elle prenne un std::span<const int> au lieu d'un tableau et une taille. 3. Dans main, créez un tableau brut (int t[]...). 4. Appelez la fonction. Remarquez que vous n'avez plus besoin de passer la taille manuellement !

    Exercice 17 : Optimisation avec Référence Constante - Objectif : Éviter les copies inutiles sur les gros objets.
    - Tâche : 1. Écrivez une fonction afficher_message(std::string message) (par copie). 2. Modifiez-la pour utiliser le passage par référence constante (const std::string&). 3. Ajoutez un cout dans la fonction pour vérifier que le message s'affiche bien. Question : À l'utilisation dans le main, est-ce que le code d'appel change ? (Réponse attendue : Non, c'est transparent, mais plus rapide).

    Exercice 18 : Vecteur Dynamique et Span - Objectif : Utiliser un tableau qui peut grandir (vector) avec une fonction moderne.
    - Tâche : 1. Incluez <vector>. 2. Créez un std::vector<int> vide. 3. Utilisez une boucle while pour demander des nombres à l'utilisateur. Ajoutez-les au vecteur avec .push_back(). Arrêtez quand il tape -1. 4. Appelez votre fonction calculer_somme (celle avec le std::span de l'exercice 16) en lui passant ce vecteur. Observation : Notez que le span accepte aussi bien le tableau brut (Ex 16) que le vecteur (Ex 18).
